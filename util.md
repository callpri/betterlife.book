# 工具集

工具集分为三类: 通用函数、工具类和api工具。

- 通用函数

  - 路径: core/include/

- 工具类

  - 路径: core/util/

- api工具

  - 路径: api/common/
  - 路径: api/wechat/

### 通用函数

* common.php    : 常用通用函数
* compatible.php: 兼容函数
  - 支持低于PHP 5.3版本的相同功能,如get_called_class。
* shortcut.php  : 缩写函数快速调用
* system.php    : 通用系统函数
* thirdparty    : 第三方通用函数工具集

### 工具集

工具集目录列表如下

* common  : 常用工具类目录
* db      : 数据库目录
* office  : word, excel读写工具目录
* datetime: 日期时间类目录
* config  : 配置类目录
* email   : 电子邮件类目录
* view    : 显示工具类目录
* xml     : XML处理工具类目录
* mobile  : 手机工具类目录
* net     : 网络工具类目录
